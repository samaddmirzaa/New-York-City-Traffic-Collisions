# NYC Traffic Collision Analysis (2021–2023)


An exploratory analysis of 238,421 traffic collisions recorded across New York 
City's five boroughs between January 2021 and April 2023. The project uses 
raw NYPD Motor Vehicle Collisions data to identify patterns in collision 
frequency, severity, timing, location, and contributing factors.

## Findings
- **635 people killed** and **116,226 injured** over the study period
- **Brooklyn** recorded the most collisions of any borough (76,416)
- **Pedestrians** account for the majority of traffic fatalities
- **Belt Parkway** was the highest-collision street with 3,728 incidents
- **Driver inattention** was the leading contributing factor
- Collisions peak between **3pm and 6pm** on weekdays
 
## Data Source
NYC Open Data — NYPD Motor Vehicle Collisions  

All done in Microsoft Excel
